Tic-Tac-Toe [ Java and JavaFx]

Description
  • A multilevel Tic-Tac-Toe game against computer, includes a scoreboard for top 3 scores
  • Made a Networking program to connect each client with the server and let each client play against it
  • Applied Min-Max algorithm for setting up the difficulty and sorting algorithms for scoreboard
  • Applied OOP the Array-list data structure to store each game played by a client against the server
